# Dubai Mall Digideck

An interactive, cinematic digideck built to showcase Dubai Mall as a premium destination for retail brands, partners, and sponsors.

---

## Live Demo

https://liatai-digideck.vercel.app/

---

## Objective

To create a modern alternative to traditional sales decks by building a web-based, interactive experience that communicates scale, luxury, and opportunity.

---

## Features

- Cinematic hero section with video background
- Animated statistics section
- Smooth transitions and motion effects
- Luxury-focused UI design
- Responsive layout

---

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Vercel

---

```bash
git clone https://github.com/Nmn242002/LIAT.ai-project-digideck.git
